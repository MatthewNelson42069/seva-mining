This plan adds a prompt instruction (with article-style framing), a post-validation guard in _research_and_draft, an instance counter, and a run-end log line. The thread bullet is also rewritten to emphasize fact-stringing narratives. Task 2 adds 4 tests covering the floor boundary.

Purpose: Prevent undersized long_form posts from reaching the dashboard. Improve format selection accuracy.
Output: Updated content_agent.py (prompt rewrites for both thread and long_form + validation + counter + log) and 4 new tests.
</objective>
