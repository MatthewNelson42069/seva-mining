2. `async def _assemble_digest(self, session: AsyncSession) -> dict`:
   - Compute yesterday boundaries:
     ```python
     today = date.today()
     yesterday_start = datetime.combine(today - timedelta(days=1), datetime.min.time()).replace(tzinfo=timezone.utc)
     yesterday_end = datetime.combine(today, datetime.min.time()).replace(tzinfo=timezone.utc)
     ```
   - **Yesterday approved count** (SENR-08 — reads from existing status/decided_at fields):
     Query `SELECT COUNT(*) FROM draft_items WHERE status IN ('approved', 'edited_approved') AND decided_at >= yesterday_start AND decided_at < yesterday_end`
   - **Yesterday rejected count** (SENR-08):
     Query `SELECT COUNT(*) FROM draft_items WHERE status = 'rejected' AND decided_at >= yesterday_start AND decided_at < yesterday_end`
   - **Yesterday expired count**:
     Query `SELECT COUNT(*) FROM draft_items WHERE status = 'expired' AND updated_at >= yesterday_start AND updated_at < yesterday_end`
   - **Top 5 stories** from yesterday (by score):
     Query `SELECT * FROM draft_items WHERE created_at >= yesterday_start ORDER BY score DESC NULLS LAST LIMIT 5`
     For each item, build: `{"headline": first_sentence_of_rationale, "source_account": item.source_account, "platform": item.platform, "score": float(item.score or 0), "source_url": item.source_url}`
   - **Queue snapshot** (current pending):
     Query `SELECT platform, COUNT(*) FROM draft_items WHERE status='pending' GROUP BY platform`
     Build: `{"twitter": N, "instagram": N, "content": N, "total": sum}`
   - **Priority alert** (highest-scoring currently-pending item):
     Query `SELECT * FROM draft_items WHERE status='pending' ORDER BY score DESC NULLS LAST LIMIT 1`
     Build: `{"id": str(item.id), "platform": item.platform, "score": float(item.score or 0), "source_account": item.source_account, "headline": first_sentence_of_rationale, "expires_at": item.expires_at.isoformat() if item.expires_at else None, "source_url": item.source_url}`
   - Return full digest dict
