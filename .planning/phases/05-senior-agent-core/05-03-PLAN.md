1. `async def _enforce_queue_cap(self, session: AsyncSession, new_item_id: uuid.UUID) -> None`:

   - Count pending items excluding `new_item_id`: `select(func.count()).select_from(DraftItem).where(DraftItem.status == "pending", DraftItem.id != new_item_id)`
   - If count < cap: return (queue not full)
   - Get new item: `await session.get(DraftItem, new_item_id)`
   - Get lowest-scoring existing item: `select(DraftItem).where(status=="pending", id != new_item_id).order_by(DraftItem.score.asc(), DraftItem.expires_at.asc()).limit(1).with_for_update(skip_locked=True)`
   - If `float(new_item.score or 0) > float(lowest.score or 0)`: delete lowest (new item stays)
   - Else: delete new_item (new item loses)
