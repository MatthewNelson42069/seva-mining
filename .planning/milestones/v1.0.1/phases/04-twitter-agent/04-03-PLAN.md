```
- NEVER mention "Seva Mining" or any variant of the company name

```

User prompt:
```
Original tweet by @{author_handle}:
"{tweet_text}"

1. "reply_alternatives": An array of 3 reply drafts responding to this tweet
2. "rt_alternatives": An array of 3 retweet-with-comment drafts
3. "rationale": A 1-2 sentence explanation of why this tweet matters and what angle the drafts take

Return ONLY valid JSON, no markdown.
```

Parse the JSON response. If JSON parsing fails, log error and return empty dict. If fewer than 2 alternatives returned for either type, that is acceptable (2-3 range per TWIT-08).

**2. `async def _check_compliance(self, draft_text: str) -> bool`:**

Call `self.anthropic.messages.create` with model `claude-3-haiku-20240307` (cheapest, fastest):

```
Does the following text mention 'Seva Mining' (or any variant like 'Seva', 'seva mining', 'SEVA') OR contain financial advice (buy/sell/invest recommendations, phrases like 'should buy', 'consider investing', 'great investment', 'price target')? Answer only YES or NO.

Text: {draft_text}
```

Parse response: `answer.strip().upper()`. Return `True` if answer is `"NO"` (passes compliance). Return `False` if `"YES"` or any other response (fail-safe: ambiguous = block).

**3. `async def _process_drafts(self, session: AsyncSession, qualifying_posts: list[dict]) -> tuple[int, int, list]`:**

For each qualifying post:
1. Call `_draft_for_post(post)` to get alternatives + rationale
2. For each alternative text (reply + RT alternatives combined): call `_check_compliance(text)`
3. Drop alternatives that fail compliance, log each failure to an errors list
4. If at least 1 alternative survives compliance in either category:
   - Create a `DraftItem` record:
     ```python
     DraftItem(
         platform="twitter",
         status="pending",
         source_url=f"https://x.com/i/web/status/{tweet_id}",
         source_text=tweet_text,
         source_account=author_handle,
         follower_count=author_followers,
         score=composite_score,
         alternatives=[
             {"type": "reply", "text": alt_text}
             for alt_text in passing_reply_alts
         ] + [
             {"type": "retweet_with_comment", "text": alt_text}
             for alt_text in passing_rt_alts
         ],
         rationale=rationale,
         urgency="high" if composite_score > 0.7 else "medium",
         expires_at=tweet_created_at + timedelta(hours=6),
         engagement_snapshot={
             "likes": likes,
             "retweets": retweets,
             "replies": replies,
             "views": views,
             "captured_at": datetime.now(timezone.utc).isoformat(),
         },
     )
     ```
   - Add to session, increment items_queued counter
5. If ALL alternatives fail compliance for a post, skip it entirely, increment items_filtered, log "all alternatives failed compliance for tweet {id}"
6. Return (items_queued, items_filtered_by_compliance, errors_list)
