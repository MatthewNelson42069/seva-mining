2. `async def _check_breaking_news_alert(self, session: AsyncSession, item_id: uuid.UUID) -> None`:
   - Replace the stub from Plan 03
   - Get item via `session.get(DraftItem, item_id)`
   - Read threshold from config key `senior_breaking_news_threshold` (default "8.5")
   - If `float(item.score or 0) >= threshold`:
     - Extract headline: first sentence of `item.rationale` (split on ". " and take [0] + ".")
     - Read dashboard URL from config key `dashboard_url` (default "https://app.sevamining.com")
     - Call `await send_whatsapp_template("breaking_news", {"1": headline, "2": item.source_account or "unknown", "3": str(round(float(item.score), 1)), "4": dashboard_url})`
     - Wrap in try/except, log warning on failure, do NOT re-raise

3. `async def _check_expiry_alerts(self, session: AsyncSession) -> None`:
   - Read `senior_expiry_alert_score_threshold` (default "7.0") and `senior_expiry_alert_minutes_before` (default "60")
   - Query: pending items where `score >= threshold` AND `expires_at BETWEEN now() AND now() + timedelta(minutes=minutes_before)` AND `alerted_expiry_at IS NULL`
   - For each candidate:
     - Minutes remaining: `int((item.expires_at - datetime.now(timezone.utc)).total_seconds() / 60)`
     - Read dashboard URL from config
     - Call `await send_whatsapp_template("expiry_alert", {"1": item.platform or "unknown", "2": headline_from_rationale, "3": str(minutes_remaining), "4": dashboard_url})`
     - Set `item.alerted_expiry_at = datetime.now(timezone.utc)`
     - Wrap each send in try/except

4. `async def _send_engagement_alert(self, item: DraftItem, dashboard_url: str) -> None`:
   - Extract excerpt: first 100 chars of `item.source_text` (truncate at word boundary if needed)
   - Compute engagement score from snapshot: `likes * 1 + retweets * 2 + replies * 1.5`
   - Call `await send_whatsapp_template("breaking_news", {"1": excerpt, "2": item.source_account or "unknown", "3": str(round(engagement_score, 1)), "4": dashboard_url})`
   - Wrap in try/except

5. `async def _check_engagement_alerts(self, session: AsyncSession) -> None`:
   - Load watchlist account handles: `select(Watchlist.account_handle).where(Watchlist.platform == "twitter", Watchlist.active == True)`
   - Build set: `watchlist_handles = {row[0].lstrip("@").lower() for row in result}`
   - Query candidates: pending twitter items where `engagement_alert_level != "viral"` (or is None)
     Use: `select(DraftItem).where(DraftItem.status == "pending", DraftItem.platform == "twitter").where(or_(DraftItem.engagement_alert_level.is_(None), DraftItem.engagement_alert_level == "watchlist"))`
   - Read dashboard URL from config
   - For each item:
     - Extract likes and views from `engagement_snapshot` JSONB: `snap = item.engagement_snapshot or {}; likes = snap.get("likes", 0) or 0; views = snap.get("views", 0) or 0`
     - Determine `is_watchlist = (item.source_account or "").lstrip("@").lower() in watchlist_handles`
     - Current level = `item.engagement_alert_level`
     - If `is_watchlist`:
       - If `current_level is None and likes >= 50 and views >= 5000`: send alert, set level to `"watchlist"`
       - Elif `current_level == "watchlist" and likes >= 500 and views >= 40000`: send alert, set level to `"viral"`
     - Else (non-watchlist):
       - If `current_level is None and likes >= 500 and views >= 40000`: send alert, set level to `"viral"`

6. `async def run_expiry_sweep(self) -> None`:
   - Create session via `async with AsyncSessionLocal() as session:`
   - Create AgentRun with `agent_name="expiry_sweep"`, `started_at`, `status="running"`
   - Step 1: Bulk-expire stale items: `update(DraftItem).where(DraftItem.status == "pending", DraftItem.expires_at != None, DraftItem.expires_at < func.now()).values(status="expired", updated_at=func.now())`
   - Capture `result.rowcount` as items_found (expired count)
   - Step 2: `await self._check_expiry_alerts(session)`
   - Step 3: `await self._check_engagement_alerts(session)`
   - On success: `run.status = "completed"`, `run.ended_at = now`
   - On exception: `run.status = "failed"`, `run.errors = [str(exc)]`, do NOT re-raise
   - Always commit in finally block
